An image viewer for viewing

No toolbars or distractions-pure space efficiency

Beautifully simple

qView was designed from the very start to be as visually minimal and space efficient as possible. No cluttered interface, just your image and a titlebar.

Fast and lightweight

You shouldn't have to wait just to view an imageâ€”qView opens practically instantly, and switching images is just as quick, all while having low memory and CPU usage.

Fully-featured

With features such as animated GIF controls, file history, rotation/mirroring, and multithreaded preloading, minimalism doesn't have to be a compromise.

Configurable

qView features a myriad of configurable preferences to ensure the best experience for as many people as possible.

Compatible

qView supports all common image formats including bmp, gif, jpg, png, tiff, and webp.

Cross-platform

Whatever operating system you use, the experience is designed to be as native as possible.

Free and open-source

qView is completely free and open source software based on Qt5. It can be found on GitHub under the GPLv3 license.
